# Safety Notes

DarkDrive AI Simulation is a simulation-only project for learning, experimentation, documentation, and portfolio work.

## Boundaries

- Simulation only.
- No public road testing.
- No real car control.
- No unsafe deployment.
- No bypassing of vehicle safety systems.
- No instructions for controlling a real vehicle.

## Current Scope

The current implementation focuses on:

- Computer vision experiments.
- Driving data collection in simulation.
- Lane detection prototypes.
- Behavior cloning experiments.
- Model evaluation inside simulated environments.

## Future RC Testing Note

Any future RC testing would need to be low speed, indoors or on a private track, manually supervised, and designed with an emergency stop. RC testing is not part of the current implementation.
